	·Como hacer una compra
1.- Al ingresar a la opción de compras, el usuario puede ver todo el catalogo de productos dando 
	clic al botón Buscar Todo,
	y así ingresar en la caja de texto el id del producto que se comprará, posteriormente seleccionar
	la opción "Seleccionar por ID" para tomar los datos y después modificar la cantidad a
	comprar. Después se tomará la opción de "Guardar Compra" para pasar a otro producto, al
	finalizar la compra se debe de confirmar en el botón "Finalizar Compra".
	·Como hacer una venta
1.- Al ingresar a la opción de ventas, el usuario puede ver todo el catalogo de productos dando
	clic al botón Buscar Todo,
	y así ingresar en la caja de texto el id del producto que se venderá, posteriormente seleccionar
	la opción "Seleccionar por ID" para tomar los datos y después modificar la cantidad a
	vender. Después se tomará la opción de "Guardar Venta" para pasar a otro producto, al
	finalizar la venta se debe de confirmar en el botón "Finalizar Venta".
