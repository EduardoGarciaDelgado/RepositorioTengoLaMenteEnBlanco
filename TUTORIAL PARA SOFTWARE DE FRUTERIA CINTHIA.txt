	·Exportacion de la Base de Datos

	(esto con el fin de exportar la Base de Datos).
	·Como hacer una compra
1.- Al ingresar a la opcion de compras, el usuario puede buscar todo el catalogo de productos
	y asi ingresar en la caja de texto el id del producto que se comprara, se debe seleccionar
	la opcion "Seleccionar por ID" para tomar los datos y despues modificar la cantidad a
	comprar. Despues se tomara la opcion de "Guardar Compra" para pasar a otro producto, al
	finalizar la compra se debe de confirmar en el boton "Finalizar Compra".
	·Como hacer una venta
1.- Al ingresar a la opcion de ventas, el usuario puede buscar todo el catalogo de productos
	y asi ingresar en la caja de texto el id del producto que se vendera, se debe seleccionar
	la opcion "Seleccionar por ID" para tomar los datos y despues modificar la cantidad a
	vender. Despues se tomara la opcion de "Guardar Venta" para pasar a otro producto, al
	finalizar la venta se debe de confirmar en el boton "Finalizar Venta".
